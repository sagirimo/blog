# 第十章-分子与晶体结构

# 10.1-离子键

# 离子键及其特点

定义：正负离子间的静电作用 特点：没有方向性/饱和性、本质是静电力、极性与元素电负性有关。

# 离子的特征

离子的电荷：原子在形成离子化合物过程中失去或得到的电子数。正负往往和原子的电子构型有关

离子半径：离子晶体中正负离子的接触半径。

# 离子键强度与晶格能

晶格能（U）（Lattice Enthalpies） 定义：晶格能表示相互远离的气态正离子

Born-Lande公式 U=-Ve∝Z1Z2/r 其中，Ve为正负离子间吸引力和排斥力达平衡时，体系的位能；Z1、Z2分别为正负离子的电荷数，r为正负离子间距。

离子化合物的性质

  • 高熔点 High melting points
  • 高沸点 High boiling points
  • 易脆性 Brittleness
  • 溶解性 Some are soluble,some not

然而，离子键不能解释全部问题，故出现共价键学说

# 10.2-经典Lewis学说

共价键（Lewis） 定义：同种原子以及电负性相近的原子间可以通过共用电子对形成分子时，通过共用电子对形成的化学键。 PS.其形成的分子称为共价分子。

# 10.3-价电子对互斥理论和分子构型

# 价电子互斥理论的要点

1.价电子对 2.价电子对的空间排布 3.分子或离子的几何构型 4.键对只包括形成σ键而不包括形成π键的电子对

# 分子的几何构型

价电子数目为1——原子 价电子数目为2——直线型 价电子数目为3——平面三角型 价电子数目为4——正四面体型 价电子数目为5——三角双锥型 价电子数目为6——正八面体型

一些例子： SF4 ClF3 XeF2 XeF4 SOF2及SO2Cl2

# 10.4-价键理论

# H~2~分子共价键的形成

# 共价键的本质和特征

本质：原子轨道重叠成键。 原子轨道重叠，核间电子云密度增大，降低了体系的能量。 原子轨道重叠的越多，两核间电子云密度越大，形成的共价键就越稳定。 价键理论认为，共价键本质是原子相互靠近时原子轨道发生重叠（即波函数叠加），原子间通过共用自旋相反的电子对成键

特征：具有饱和性和方向性

1.饱和性 每个原子提供的成键（原子）轨道数和形成分子时可提供的未成对电子数是一定的

2.方向性 p、d、f等轨道在空间均具有一定的取向

# 10.7-分子间作用力与氢键

# 分子间作用力

一般没有方向性/饱和性

1.取向力

极性分子是偶极子，其偶极称为固有偶极，有正负两极。不同分子相互靠近时同斥异吸，具有一定的取向性。

2.诱导力

极性分子和非极性分子之间存在。极性分子的极性诱导非极性分子变形，产生诱导偶极。越大越容易变形，越小越难变形

3.色散力

电子概率运动，原子核也不停振动，在瞬时可以产生瞬时偶极，继而为其他分子产生瞬时诱导偶极。分子量越大色散力越大，线性分子力更大（排列更整齐）。

范德华力很弱。化学键一般在200-300kJ每mol，而范德华力约在2-20。

# 氢键

强于范德华力（分子间作用力），弱于化学键

1.氢键的生成

发现：卤氧氮的氢化物沸点反常现象。

强大的电负性使得H的电子偏移，形成裸露的质子，进而吸引高电负性的粒子。

2.氢键的本质与特点

饱和性，水能形成4个，一个O两个，两个H一个。

3.氢键的键长和键能

键能小，键长长。

F/O/N成氢键强度递减。

4.氢键的种类

分子间氢键和分子内氢键

分子间氢键使熔沸点升高，水中溶解度增大，

分子内氢键使熔沸点降低，水中溶解度减小。

5.氢键对物质性质的影响

1）熔沸点升高（分子间） （注意看氢键个数与氢键强度） 同理，因为熔化热、气化热、蒸气压与熔沸点有关，也会相应变化。

2）对水/冰密度的影响

3）对物质溶解度的影响

4）对蛋白质构型的影响

5）对物质酸性的影响
