# 10.1-离子键

# 1.离子键及其特点

定义：正负离子间的静电吸引力叫离子键。 特点：既没有方向性，也不具有饱和性。

# 2.离子的特征
# 3.离子键强度与晶格能

晶格能（U）（Lattice Enthalpies） 定义：晶格能表示相互远离的气态正离子

Born-Lande公式 U=-Ve∝Z1Z2/r 其中，Ve为正负离子间吸引力和排斥力达平衡时，体系的位能；Z1、Z2分别为正负离子的电荷数，r为正负离子间距。

离子化合物的性质

  • 高熔点 High melting points
  • 高沸点 High boiling points
  • 易脆性 Brittleness
  • 溶解性 Some are soluble,some not
# 然而，离子键不能解释全部问题，故出现共价键学说

# 10.2-经典Lewis学说

共价键（Lewis） 定义：同种原子以及电负性相近的原子间可以通过共用电子对形成分子时，通过共用电子对形成的化学键。 PS.其形成的分子称为共价分子。

# 10.3-价电子对互斥理论和分子构型

# 1.价电子互斥理论的要点

1.价电子对 2.价电子对的空间排布 3.分子或离子的几何构型 4.键对只包括形成σ键而不包括形成π键的电子对

# 2.分子的几何构型

价电子数目为1——原子 价电子数目为2——直线型 价电子数目为3——平面三角型 价电子数目为4——正四面体型 价电子数目为5——三角双锥型 价电子数目为6——正八面体型

一些例子： SF4 ClF3 XeF2 XeF4 SOF2及SO2Cl2

# 10.4-价键理论

# 1.H~2~分子共价键的形成
# 2.共价键的本质和特征

本质：原子轨道重叠成键。 原子轨道重叠，核间电子云密度增大，降低了体系的能量。 原子轨道重叠的越多，两核间电子云密度越大，形成的共价键就越稳定。 价键理论认为，共价键本质是原子相互靠近时原子轨道发生重叠（即波函数叠加），原子间通过共用自旋相反的电子对成键

特征：具有饱和性和方向性

1.饱和性 每个原子提供的成键（原子）轨道数和形成分子时可提供的未成对电子数是一定的

2.方向性 p、d、f等轨道在空间均具有一定的取向
